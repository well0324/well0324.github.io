來規劃將有三個部分:

1. 會員的頭銜

根據積分機制得到不同門檻之頭銜，例如: 素人 / 樂粉 / 村長 / 田橋 / 莊主 / 土豪 / 城主 / 領主 / 國王…(名稱思考中..)。

2. 會員的積分

與積分獎勵機制連動。也就是積分獎勵機制的累積，來自至少包含連續登入獎勵、留言、投稿、按讚、被按讚、分享、被分享等這些行為，這些行為都可以累積一定數量積分，串接GASH會員紅利。

3. 會員的貢獻值

統計會員投稿內容篇數以及累積流量貢獻值，後續串接對投稿者的獎勵機制。
